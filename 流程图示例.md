# 流程图示例

## 用户登录流程

```mermaid
flowchart TD
    A[打开应用] --> B{是否已登录}
    B -->|是| C[进入首页]
    B -->|否| D[显示登录页]
    D --> E[输入手机号]
    E --> F[获取验证码]
    F --> G{验证码是否正确}
    G -->|是| C
    G -->|否| H[提示错误，重新输入]
    H --> E
```

## 结构说明

| 元素 | 写法 | 用途 |
|------|------|------|
| `A[文本]` | 方括号 | 流程节点（矩形） |
| `B{文本}` | 花括号 | 判断/分支（菱形） |
| `-->` | 实线箭头 | 流程走向 |
| `-->|条件|` | 带标签箭头 | 条件分支说明 |

> 更多语法参考：[Mermaid 官方文档](https://mermaid.js.org/syntax/flowchart.html)
